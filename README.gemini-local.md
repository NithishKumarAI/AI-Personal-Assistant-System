# AI Diary Assistant — `gemini-local`

Local runtime with **Groq Whisper** + **Gemini** + **Notion** via `.env`.

**Full guide:** [`gemini-local/README.md`](https://github.com/NithishKumarAI/AI-Personal-Assistant-System/blob/gemini-local/README.md)

## Quick Start

```bash
git checkout gemini-local
pip install -r requirements.txt
copy .env.example .env
streamlit run app.py
```
